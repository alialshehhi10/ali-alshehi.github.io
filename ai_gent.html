Intelligent Agents — July 2025 B

This module provides a critical understanding of intelligent agent theory, tools, and programming techniques—enabling the design, development, and deployment of agent systems using languages like Java and Python.

Learning Outcomes

  • Identify and critically analyse agent-based systems, differentiating between architectures and approaches.
  • Apply and evaluate intelligent agent techniques to real-world problems where risk and uncertainty are involved.
  • Deploy appropriate tools and skills for designing and implementing agent-based systems, considering legal, social, ethical and professional issues.
  • Develop team skills for effective collaboration in virtual professional environments.

Unit 1: Introduction to Agent-Based Computing

Learning Outcomes

  • Understand the role of intelligent agents in industry.
  • Identify the skills needed to be proficient in agent-based systems.
  • Recognise factors impacting ML/agents and ways to address them.
  • Understand high-level categories relevant to agent systems.

Unit 2: Introducing First Order Logic

Learning Outcomes

  • Understand the core elements of first-order logic.
  • Create and reason over first-order logic.

Unit 3: Agent Architectures

Learning Outcomes

  • Critically evaluate a range of agent architectures.
  • Select an appropriate architecture for a given task.
  • Explain the difference between intentions and desires.

Unit 4: Hybrid Agent Architectures

Learning Outcomes

  • Critically assess agent architectures.
  • Justify the selection of an appropriate architecture to solve a given problem.

Unit 5: Agent Communication

Learning Outcomes

  • Develop and work with ontologies.
  • Create inter‑agent communications using relevant languages.

Unit 6: Working Together

Learning Outcomes

  • Develop dialogues using KQML/KIF or other agent communication languages.
  • Understand deploying ontologies for knowledge sharing.

Unit 7: Natural Language Processing (NLP)

Learning Outcomes

  • Evaluate challenges in developing and deploying NLP systems.
  • Understand core principles of NLP technologies.

Unit 8: Understanding NLP

Learning Outcomes

  • Explain key elements of NLP models.
  • Engage with worked examples of common NLP methods.
  • Develop parse trees to support NLP understanding.

Unit 9: Introduction to Adaptive Algorithms

Learning Outcomes

  • Understand the core concepts of artificial neural networks.
  • Appraise relative strengths of adaptive techniques.

Unit 10: Deep Learning in Action

Learning Outcomes

  • Understand the development of technology and its limitations.
  • Appraise products/services for socio‑economic and ethical impacts.
  • Explain the importance of data in delivering these technologies.

Unit 11: Intelligent Agents in Action

Learning Outcomes

  • Apply concepts and theories to specific sectors.
  • Understand how technology can drive efficiency.
  • Evaluate the merits of new approaches in context.

Unit 12: The Future of Intelligent Agents

Learning Outcomes

  • Appraise current technologies.
  • Assess potential evolution based on today’s examples.
  • Evaluate potential ethical and social considerations.
